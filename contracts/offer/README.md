# KNOXBOTS MaintainME Brochure

This is a modern, professional brochure for the KNOXBOTS MaintainME service plan built with React and Tailwind CSS.

## Setup Instructions

1. Make sure you have Node.js installed (v14 or higher recommended)

2. Install the required dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm start
   ```

4. To build for production:
   ```
   npm run build
   ```

## Features

- Professional design with KNOXBOTS branding
- Responsive layout that works on all devices
- Print-optimized stylesheet for physical handouts
- Interactive elements for digital viewing
- Clear comparison between standard warranty and MaintainME coverage
- Detailed benefits and pricing information

## Printing Instructions

To print the brochure:
1. Open the application in a modern browser (Chrome recommended)
2. Click the "Print Brochure" button in the bottom right corner
3. In the print dialog, set the following options:
   - Paper size: Letter
   - Scale: Default (100%)
   - Margins: Default
   - Background graphics: Enabled
4. Click Print

## Customization

- Color scheme can be adjusted in the `tailwind.config.js` file
- Content can be modified directly in the `knoxbots-maintainme.jsx` file
- Additional styling can be added to `index.css`
